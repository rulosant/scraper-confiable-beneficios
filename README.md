# scraper-confiable-beneficios
Scraper que extrae la lista de comercios adheridos a una campaña es Tarjeta Confiable BPN


## Instalacion

```bash
git clone <este_repo>

pip install -r requirements.txt
```
Previamente se puede crear un  virtualenv para descargar los paquetes en ese entorno


## Uso
```bash
python3 club.py
```

Al ejecutar, mostrará la lista de campañas activas en el sitio BPN .com .ar. 
Permite elegir una campaña, y luego mostrará los comercios adheridos a esa campaña en la Ciudad de Neuquén
